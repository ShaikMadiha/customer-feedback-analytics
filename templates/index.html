🚀 About This Project

Customer Feedback Analytics is an AI-powered web application that analyzes customer reviews using a fine-tuned DistilBERT transformer model. The application classifies each review as Positive, Neutral, or Negative and provides confidence scores along with class probabilities to help users better understand customer sentiment. The project is built with Flask for the backend and features a clean, responsive interface with Light and Dark mode support.

🤖

DistilBERT Model

Fine-tuned transformer model for three-class sentiment analysis.

⚡

Real-Time Prediction

Analyzes customer reviews instantly and returns sentiment with confidence.

📊

Prediction Details

Displays sentiment, confidence score, and probability distribution for every review.

🌙

Modern User Interface

Responsive web interface with Light/Dark mode for a better user experience.

Built with ❤️ using Flask, DistilBERT, Hugging Face Transformers and PyTorch.
